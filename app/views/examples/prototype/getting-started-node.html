Prerequisites

In order to use the prototype kit, you must have installed the following on your development machine

  • Node & NPM
  • Text editor of your choice

Installation

Node

Once you have Node & NPM installed on your machine you're ready to download the prototype kit. This can be done by either downloading the project as a zip, or by forking the project. After the download has finished, move the project to a suitable place on your computer, then open a terminal and navigate to it and run the following command:

npm install

This will install all of the depenancies that the prototype kit requires to run, includuing the DVLA internal frontend kit.

Running

Node

Now that you have the prototype kit installed and setup, you can verify everything works by starting your prototype. To do so, open your terminal and run the following command:

npm start

So long as everything happened correctly, you should now be running your prototype app. To view it, vist localhost:3000 in your browser.
